📊 Customer Behavior Analysis
📌 Project Overview

This project analyzes customer shopping behavior using Python, SQL, and Power BI. The goal is to extract insights about customer demographics, purchasing patterns, and revenue trends.

🛠 Tools Used
Python (Pandas)
PostgreSQL (SQL)
Power BI
📈 Key Insights
Revenue distribution by subscription status
Top-selling products per category
Customer segmentation by age group
Spending patterns across different demographics
📂 Files
CUST.ipynb → Data cleaning & preprocessing
customer_behave.sql → SQL queries
cust behavior dashboard.pbix → Power BI dashboard
